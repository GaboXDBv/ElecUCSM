Fundamentos de Lógica Difusa

Modelado de la incertidumbre y sistemas no lineales

Historia y Conceptos

El Origen

Introducida por Lotfi A. Zadeh (1965) para formalizar el razonamiento humano. A diferencia de la lógica booleana (0 o 1), la lógica difusa permite grados de veracidad entre ambos valores.

Ventajas Clave

  • Flexibilidad y tolerancia a la imprecisión.
  • Capacidad para modelar sistemas altamente no lineales.
  • Basada en lenguaje natural (palabras en lugar de números complejos).
Definiciones Fundamentales

Universo de Discurso (X)

Es el rango total de valores posibles para una variable de entrada. Por ejemplo, si medimos temperatura, X podría ser [0°C, 100°C].

Conjunto Difuso (A)

Clase de objetos con un grado de pertenencia continuo entre 0 y 1.

f_A(x) : X → [0, 1]

Operaciones Lógicas

Intersección (AND)

f_(A ∩ B)(x) = Min[f_A(x), f_B(x)]

Unión (OR)

f_(A ∪ B)(x) = Max[f_A(x), f_B(x)]

Complemento (NOT)

f_A'(x) = 1 - f_A(x)
Funciones de Pertenencia (MFs)

Son curvas que definen cómo cada punto de entrada se mapea a un valor de pertenencia.

Triangular

Definida por 3 parámetros (a, m, b). Es la más común por su bajo costo computacional.

Trapezoidal

Útil cuando existe un rango de valores que tienen pertenencia total (1) al conjunto.

Gaussiana

Proporciona transiciones muy suaves. Ideal para sistemas de control que requieren alta precisión y suavidad.

Parámetros de la Función μ

Soporte

Intervalo del dominio donde la pertenencia es mayor que cero.

sop(x) = { x ∈ X | μ_A(x) > 0 }

Altura

Es el valor máximo de pertenencia alcanzado por la función. Si la altura es 1, el conjunto se llama "normal".
