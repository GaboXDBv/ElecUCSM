Marco teórico: cómo crear funciones en Python

Una función es un bloque de código reutilizable que realiza una tarea específica. Sirve para organizar mejor los programas, evitar repetir código y facilitar la lectura.

  • Se define con la palabra reservada def.
  • Puede recibir parámetros, que son datos de entrada.
  • Puede retornar un valor usando return.
  • Puede no retornar nada y solo mostrar resultados con print().
  • Debe tener buena indentación para que Python reconozca el bloque.
def saludar(nombre):
    return f"Hola, {nombre}"

En este ejemplo, saludar es el nombre de la función y nombre es el parámetro. Cuando la función se ejecuta, devuelve un saludo personalizado.

Partes importantes de una función

  • Nombre: debe describir lo que hace la función.
  • Parámetros: valores que la función necesita para trabajar.
  • Cuerpo: instrucciones que ejecuta la función.
  • Return: resultado final que entrega la función.

Buenas prácticas

  • Usar nombres claros como calcular_area o es_par.
  • Hacer una sola tarea por función.
  • Retornar resultados en lugar de depender siempre de print().
  • Probar la función con varios casos.
